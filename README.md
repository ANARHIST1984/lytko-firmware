# Lytko — прошивки для термостатов

Здесь публикуются обновления прошивок для устройств Lytko.

## Устройства

| Устройство | Описание |
|------------|----------|
| Lytko 100 | Термостат с круглым дисплеем |
| Lytko NS | Термостат для скрытого монтажа или DIN-рейки |
| Дисплей Lytko | Круглый дисплей для Lytko 100 |

## Что нового

Описание изменений доступно в каждом релизе — [смотреть все версии](../../releases)

## Как обновить прошивку

### Способ 1 — Через веб-интерфейс (рекомендуется)

1. Откройте в браузере `http://<ip термостата>/manual_update`
2. Загрузите файл `.bin` из раздела [Releases](../../releases)
3. Дождитесь перезагрузки

### Способ 2 — Через Recovery

1. Зажмите кнопку на термостате
2. Подайте питание, подождите 5–6 секунд, отпустите
3. Термостат создаст Wi-Fi сеть **Device Recovery Mode** — подключитесь к ней
4. После подключения откроется окно восстановления. Если не открылось — введите в браузере `192.168.4.1`
5. Загрузите файл `.bin` через **Download firmware**
6. Нажмите **Reboot**
