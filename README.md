# Lytko — прошивки для термостатов

Здесь публикуются обновления прошивок для устройств Lytko.

## Устройства

| Устройство | Описание |
|------------|----------|
| Lytko 100 | Термостат с круглым дисплеем |
| Lytko NS | Термостат без экрана (DIN-рейка) |

## Как обновить прошивку

### Способ 1 — Через веб-интерфейс (рекомендуется)

1. Откройте в браузере адрес термостата (например `lytko-101ns.local`)
2. Перейдите в **Настройки → Об устройстве → Обновление**
3. Загрузите файл `.bin` из раздела [Releases](../../releases)
4. Дождитесь перезагрузки

### Способ 2 — Через Recovery

1. Зажмите кнопку на термостате
2. Подайте питание, подождите 5–6 секунд, отпустите
3. Подключитесь к Wi-Fi сети **Device Recovery Mode**
4. Откройте `192.168.4.1` в браузере
5. Загрузите файл `.bin` через **Download firmware**
6. Нажмите **Reboot**
